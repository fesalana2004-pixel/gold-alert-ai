# Gold Alert AI 🥇

تطبيق ذكي لمتابعة أسعار الذهب وإرسال إشارات تداول فورية بناءً على تحليل الأخبار بالذكاء الاصطناعي.

## المميزات
- ✅ سعر الذهب لحظياً (XAU/USD)
- ✅ رسوم بيانية تفاعلية
- ✅ تحليل الأخبار بالذكاء الاصطناعي
- ✅ إشعارات فورية لإشارات التداول
- ✅ دعم 8 لغات
- ✅ مؤشر قوة الإشارة
- ✅ سجل الإشارات مع النتائج

## التقنيات المستخدمة
- HTML5, CSS3, JavaScript
- PWA (Progressive Web App)
- Chart.js للرسوم البيانية
- Service Worker للعمل بدون إنترنت
- Web Notifications API
